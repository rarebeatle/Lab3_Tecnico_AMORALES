Pasos para levantar el proyecto: 

1. Levantar los servicios: 
    a. Abrir una terminal o cdn
    b. Ubicarse en la carpeta json-server
    c. En la terminal o cdn correr el comando "npm install -g json-server"
    d.En la terminal o cdn escribir el comando "npm install"
    e. En la terminal o cdn escribir "json-server --watch db.json"

2.Levantar el proyecto: 
    a.Abrir una terminal o cdn
    b.Ubicarse en la carpeta lab3
    c.En la terminal o cdn escribir el comando "npm install"
    d.En la terminar o cdn escribir el comando "npm start"
